## 常用快捷键

删除行 `ctrl + y`

自动填写 for 方法 例如：`100.fori`

对代码重新排列格式 `Ctrl + Alt + L`

上一个方法 `alt+方向键上`

左右文件切换 `alt+方向键左右`

跳到类或方法的声明 `ctrl + B`

Ctrl + Shift +退格键（导航|最后编辑的位置）带你回到你的更改代码中的最后一个地方

实现方法 ctrl + i

构造方法 alt + insert

大小写不敏感：`file->settings->Editor->General->Code Completion -> Case sensitive completion` 设置为 `None`
